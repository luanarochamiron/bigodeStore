# bigodeStore - Gaming E-Commerce Landing Page

Uma landing page para loja virtual de jogos moderna, imersiva e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto destaca-se pelo uso de uma seção hero de tela cheia com uma imagem promocional de alto impacto do jogo *Multiversus*, aliada a uma barra de navegação superior translúcida integrada perfeitamente ao design.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do cabeçalho (`<header>`), links de navegação (`<nav>`) e blocos da seção principal de chamada.
* CSS3: Estilização completa, uso de tipografia sem serifa limpa, efeitos de transparência e gerenciamento de imagens de fundo em tela cheia.
* Flexbox: Distribuição horizontal equilibrada do cabeçalho e alinhamento dos blocos de conteúdo e botões na seção hero.

## Funcionalidades e Técnicas Aplicadas

* Barra de Navegação Translúcida: Menu superior projetado com fundo transparente, permitindo que o banner artístico da seção hero ocupe toda a viewport sem interrupções visuais agressivas.
* Alinhamento Estratégico com Flexbox: Uso de propriedades de distribuição espacial para manter o logotipo "bigodeStore" fixado à esquerda, os links de navegação (Games, Browse Games, Store) centralizados e os botões de autenticação à direita.
* Botões de Ação de Alto Contraste:
  * **Botão Principal (Sign up / Get in now!):** Preenchidos com um tom azul sólido e texto em negrito de alta visibilidade.
  * **Botão Secundário (Login):** Desenhado no formato vazado (*ghost button*) com bordas finas brancas e cantos suavizados por meio de `border-radius`.
* Design Imersivo em Tela Cheia: Otimização da imagem hero de fundo usando propriedades como `background-size: cover` para preencher as dimensões da tela de forma responsiva, conferindo uma estética premium e dinâmica voltada ao público gamer.
